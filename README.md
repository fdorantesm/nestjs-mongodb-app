### Description

A [NestJS](https://nestjs.com) API template using mongodb

### Stack

|                 | Version |
| --------------- | ------- |
| NestJS          |   9.X   |
| MongoDB         |   5.0   |
| Docker          |   20.1  |
| Docker Compose  |  1.29.2 |

### Installation

```bash
$ yarn install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
